kite (HTML Templating Engine) (opens in a new tab)

I originally made this when I was 14, back in 2015. I did not know about Angular / React / whatever popular framework was out there, so I made my own templating engine. It was super small (634b) and worked like a charm!

To get it, one would just add this script to their HTML file:

<script src="https://github.com/danilolekovic/kite.git"></script>

And then have a 'component' file like test.html:

<p>Hello world!</p>

And then, in the HTML file, you would have:

<kite src="test.html"></kite>

And it would render the component in the HTML file.
